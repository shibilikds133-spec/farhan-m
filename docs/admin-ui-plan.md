# Admin UI Architecture

## Layout
- **Sidebar Navigation:** Quick access to Overview, Members, Transactions, Settings.
- **Top Bar:** Quick actions (Export, Add Member), Notifications, Profile.

## Data Visualization
- **Cards:** Total Collected, Active Members, Pending Dues.
- **Tables:** Data tables with pagination, sorting, and inline editing for transactions.

## Security
- Protected routes using Next.js Middleware.
- Strict server-side validation for all admin actions.
