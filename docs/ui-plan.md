# UI/UX Design Plan

## Core Aesthetics
- **Theme:** Clean, modern, and trustworthy (Glassmorphism & premium UI).
- **Color Palette:**
  - Primary: Blue (#2563EB) & Navy (#063b78)
  - Accents: Emerald Green & Olive Green
  - Backgrounds: Gradients and subtle blurs (#eef7ff to white)
- **Typography:** Inter/Outfit for standard text, Cooper for branding/headers.

## Key Screens
- **Landing Page:** High-conversion hero section with interactive benefits.
- **Pay Portal:** Clean input forms with OTP/Phone verification.
- **Success/Receipt:** Celebration animations (Confetti) and instant receipt generation.
