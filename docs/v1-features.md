# Phase V1: Core Features

## Goal
Establish the foundational portal for member contributions and basic digital presence.

## Features
1. **Landing Page:** Premium UI with official branding.
2. **Authentication:** Simple 10-digit mobile number login/OTP (Inline form).
3. **Payment Collection:**
   - Support for UPI and Cash methods.
   - Admin/Member selection.
4. **Receipt Generation:**
   - Instant digital receipt creation.
   - Downloadable receipt card (html2canvas).
   - Confetti success animation.
5. **Mobile Responsiveness:** 100% mobile-first design.
