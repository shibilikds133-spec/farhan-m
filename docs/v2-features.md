# Phase V2: Advanced Features & Admin Control

## Goal
Expand the portal to support administrative operations and automated workflows.

## Features
1. **Admin Dashboard:**
   - Overview of all collections.
   - Daily/Weekly/Monthly charts and statistics.
   - Search and filter transactions.
2. **Automated Communications:**
   - WhatsApp API integration for sending receipts automatically.
   - SMS notifications for successful payments.
3. **Data Export:**
   - Download reports in Excel/CSV format.
   - Batch PDF generation.
4. **Role-Based Access Control (RBAC):**
   - Distinct views for Super Admin vs. Collection Agents.
